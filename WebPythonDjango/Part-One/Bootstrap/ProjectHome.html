Welcome to the Boostrap Project: Coffe Lover

I recreate this webpage wich contains an additional Form Page.

Above is listed the instruction steps that I take!

  1. Create a general landing page with a JumboTron
  2. Next create a NavBar on the Page that links to another html file
  3. Add two paragraphs of lorem-ipsum below the jumbotron
  4. Next use the Grid system to add in thumbnail pictures of coffee. Source: https://stocksnap.io/search/coffe. You wil need to use the thumbnail class for this. Here the image links:
    • One
    • Two
    • Three
    • Four
    • Five
    • Six
  5. Next make sure that at the largest settings you have 3 columns of pictures. On the smallest screen settings you should have 2 columns. Is up to you where you want to change.
  6. Next create a another html file for the sing up page.
  7. Create a form on this Sing Up page with Email, Password, and a Check Box.

That's it. Feel free to play around with this project and style it more to your linking. The main focuses of this project are the uses of NavBar, Jumbotron, Form, Container, and the Grid System.
